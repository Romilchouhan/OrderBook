# Core OrderBook

## Setup

1. Copy your existing C++ OrderBook code here:
   - Copy *.hpp files to include/
   - Copy *.cpp files to src/

2. Build:
   ```bash
   mkdir build && cd build
   cmake ..
   make
   ```

Your existing OrderBook implementation should include:
- Types.hpp
- Order.hpp, Order.cpp
- Limit.hpp, Limit.cpp
- OrderBook.hpp, OrderBook.cpp
- main.cpp
